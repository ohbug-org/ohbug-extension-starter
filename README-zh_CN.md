# OhbugExtension starter

使用这个模板仓库创建属于你的 Ohbug 扩展。

[English](./README.md) | 简体中文

## 开始

1.  **Git clone the repo.**
    ```shell
    git clone git@github.com:ohbug-org/ohbug-extension-starter.git
    ```
1.  **安装依赖**
    ```shell
    cd ohbug-extension-starter/
    pnpm i
    ```
1.  **开始调试**
    ```shell
    pnpm run dev
    ```
1.  **打包**
    ```shell
    pnpm run build
    ```
